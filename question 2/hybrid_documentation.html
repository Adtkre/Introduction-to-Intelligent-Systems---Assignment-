01 — Overview

What Is This System?

This project implements an Adaptive Neuro-Fuzzy Inference System (ANFIS) — the canonical hybrid of fuzzy logic and neural networks. The system takes three academic inputs, applies a fuzzy reasoning structure, then tunes all parameters automatically via neural learning, producing a precise score classified into Poor, Average, or Good.

Pure fuzzy systems require manual rule crafting and are not self-improving. Pure neural networks lack transparency. ANFIS gives you both: interpretable fuzzy IF-THEN rules that learn from data.

ANFIS 5-Layer Architecture

→
Layer 2

→
Layer 3

w̄_i = w_i / Σw_k

Layer 4

Layer 5
Output
Σ(w̄_i · f_i) weighted average
02 — Core Concept

How Fuzzy Logic & Neural Networks Integrate

ANFIS encodes a Sugeno-type fuzzy system as a differentiable feedforward network, allowing both components to coexist and complement each other:

Fuzzy Logic Provides
Structure — IF-THEN rules with linguistic labels (Low, Medium, High)

Interpretability — each rule is human-readable and auditable

Prior knowledge — domain expertise encoded as membership functions
+
Neural Network Provides
Learning — hybrid algorithm tunes MF parameters from data

Adaptability — self-corrects via backpropagation and LSE

Accuracy — gradient descent minimises prediction error over epochs

The Hybrid Learning Algorithm

Training uses a two-pass hybrid that is more efficient than pure backpropagation:

Forward Pass
Least Squares (LSE)

Premise (MF) parameters are fixed. The network identifies optimal consequent parameters (Sugeno linear coefficients) analytically via least-squares estimation — fast and exact.

Backward Pass
Backpropagation

Consequent parameters are fixed. Error gradients flow back to update the premise parameters (MF centres and widths) via steepest gradient descent.

MATLAB · ANFIS Training
%% Step 1: Generate initial FIS structure (Fuzzy Layer)
genOpt = genfisOptions('GridPartition');
genOpt.NumMembershipFunctions = [3 3 3];      % Low/Med/High per input
genOpt.InputMembershipFunctionType = 'trimf';
initFIS = genfis(trainData(:,1:3), trainData(:,4), genOpt);

%% Step 2: Attach to ANFIS trainer (Neural Layer)
trainOpt = anfisOptions();
trainOpt.InitialFIS     = initFIS;
trainOpt.EpochNumber    = 120;
trainOpt.ValidationData = testData;

%% Step 3: Hybrid training — LSE + Backpropagation
[trainedFIS, trainErr, ~, ~, testErr] = anfis(trainData, trainOpt);
%  Neural net tunes MF params  → sharper, data-fitted boundaries
%  LSE refines Sugeno coeffs   → precise linear consequents

%% Step 4: Evaluate and classify
score = evalfis(trainedFIS, [85, 78, 80]);   % → ~83 → Good
label = classifyPerf(score);                % 'Good'
