Washing Machine
Fuzzy Logic System

Dirt Level × Load Size → Optimal Cycle Time

Overview

This project implements a Mamdani fuzzy inference system (FIS) to automatically determine the optimal washing cycle time based on two linguistic input variables: Dirt Level and Load Size. The system uses 9 fuzzy IF-THEN rules and centroid defuzzification to yield a precise cycle time between 20 and 120 minutes.

MATLAB Implementation

Key code segments from washing_machine_fuzzy.m. Run the full script to generate all plots and the rule viewer.

MATLAB 
%% Create Mamdani FIS
fis = mamfis('Name', 'WashingMachine');

%% Add inputs
fis = addInput(fis, [0 10], 'Name', 'DirtLevel');
fis = addMF(fis, 'DirtLevel', 'trimf', [0  0  5],  'Name', 'Low');
fis = addMF(fis, 'DirtLevel', 'trimf', [2  5  8],  'Name', 'Medium');
fis = addMF(fis, 'DirtLevel', 'trimf', [5 10 10],  'Name', 'High');

%% Add rules (9 rules)
ruleList = [
    1 1, 1 (1) : 1;   % Low & Small   → Short
    3 3, 4 (1) : 1;   % High & Large  → VeryLong
];
fis = addRule(fis, ruleList);

%% Evaluate
result = evalfis(fis, [7, 8]);   % → ≈ 101 min

%% Open interactive viewers
ruleview(fis);
writeFIS(fis, 'WashingMachine');
